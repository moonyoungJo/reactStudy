JSX는 자바스크립트 내부에서 HTML 태그를 사용할 수 있게 해준다.
대신 바벨을 통해 자바스크립트로 변환하는 작업이 필요하다. 
사실 JSX 없이도 리액트를 사용할 수 있지만, JSX를 이용하면 더 자연스럽고
직관적인 코드로 리액트 라이브러리를 사용할 수 있다.

주의사항
-닫는 태그를 꼭 작성해야 한다
-범위 인식이 잘 안될경우 소관호로 감싸면 된다.
-여러개의 html태그를 연속적으로 작성할 수 없다.

특징
-변수의 값은 자동으로 이스케이프 처리한다.(ex <를 &lt으로 변환)

JSX가 어떻게 변환이 될까?
const e = (
    <h1>
        <p>Test</p>
    </h1>
)
var e = React.createElement(
    "h1", null, React.createElement("p", null, "Test")
)
